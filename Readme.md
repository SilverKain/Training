# Курс домашних тренировок

Интерактивный курс тренировок на 4 недели с постепенным увеличением нагрузки.

## Что внутри
- 4 недели прогрессии: адаптация → укрепление → рост объёма → уплотнение
- Тренировки по дням с упражнениями, подходами, повторениями, темпом и отдыхом
- Подсказки по группам мышц и технике выполнения каждого упражнения
- Таймер отдыха между подходами и упражнениями с рекомендациями по восстановлению
- Отслеживание прогресса: выполненные тренировки и упражнения сохраняются локально

## Онлайн-версия
- https://silverkain.github.io/Training/

## Как использовать
1. Откройте курс по ссылке выше.
2. Выберите неделю и тренировочный день.
3. Выполняйте упражнения по шагам, отмечая подходы.
4. Используйте таймер и ориентируйтесь на рекомендации по восстановлению.
